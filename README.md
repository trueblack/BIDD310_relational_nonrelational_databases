# BIDD 310: Relational and Non-Relational Databases

This repository contains coursework from BIDD 310, covering the design and implementation of both relational and non-relational database systems. 

The course explored SQL Server database development, data warehouse modeling and ETL processes, SSAS Tabular modeling, and document databases using Databricks. 

Projects span both on-premises and Azure cloud implementations, demonstrating practical skills in normalized database design, data warehousing, reporting, and modern cloud-based data storage solutions.
